# CLI Tools Launcher 🚀

Simple launcher for running multiple CLI tools from one place.

## Features

* Launch multiple scripts from a single menu
* Easy navigation
* Lightweight and fast
* Works as a standalone `.exe`

## Usage

Run the launcher and choose a tool from the list.

## Included Tools

* Currency Converter
* Guess Number Game
* Password Strength Checker
* Rock Paper Scissors
* Website Response Checker
